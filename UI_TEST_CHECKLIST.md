# 【极简转录文稿】UI 验收清单

## ✅ 视觉验收

### 1. 垂直时间线

- [ ] 时间线从列表顶部延伸到底部
- [ ] 线条位置在左侧 50px 处
- [ ] 线条宽度 2px，颜色为淡灰色
- [ ] 两端有渐变效果（透明 → 灰 → 透明）

### 2. 时间锚点

- [ ] 所有事件的时间标签对齐在时间线左侧
- [ ] 时间格式为 `MM:SS`（如 00:15）
- [ ] 字体为等宽字体（Monaco/Consolas）
- [ ] 时间圆点挂载在时间线上

### 3. 事件圆点

- [ ] 圆点直径 20px，白色背景
- [ ] 边框颜色根据风险等级变化：
  - 高风险：红色 (#f56c6c)
  - 中风险：橙色 (#faad14)
  - 低风险：绿色 (#52c41a)
- [ ] 圆点内包含模态图标（麦克风/眼睛/耳机）
- [ ] Hover 时圆点放大 1.3x
- [ ] 激活时圆点放大 1.5x + 紫色外发光

## ✅ 排版验收

### 4. Speech 事件（字幕式）

- [ ] 字号 15px（明显大于其他类型）
- [ ] 行距 1.8（充足的呼吸感）
- [ ] 无背景色（纯文本展示）
- [ ] 敏感关键词高亮（红色背景）
- [ ] 垂直间距充足（padding: 8px 0）

### 5. Visual/AudioEffect 事件（通知式）

- [ ] 字号 12px（明显小于 Speech）
- [ ] 窄版设计（max-width: 85%）
- [ ] 淡色背景（rgba 0.08 透明度）
- [ ] 左侧色条区分模态类型：
  - Visual：蓝色
  - AudioEffect：橙色
- [ ] 紧凑内边距（padding: 6px 12px）

## ✅ 交互验收

### 6. 默认状态

- [ ] 所有技术参数隐藏（置信度、强度、风险标签等）
- [ ] Speech 只显示台词文本
- [ ] Visual/Audio 只显示检测结论
- [ ] 页面整洁，无视觉噪音

### 7. 激活状态（视频播放到该事件时）

- [ ] 时间圆点放大 1.5x
- [ ] 圆点出现紫色外发光（4px 扩散）
- [ ] 内容区轻微右移（translateX 2px）
- [ ] 技术详情微动画展开：
  - Speech：情绪标签 + 风险标签
  - Visual：置信度 + 风险标签
  - AudioEffect：强度 + 风险标签
- [ ] 展开动画时长 0.3s，流畅自然

### 8. Hover 状态

- [ ] 鼠标悬停在事件上时，圆点放大 1.3x
- [ ] 无其他夸张效果（保持简约）

### 9. 点击跳转

- [ ] 点击任意事件区域，视频跳转到对应时间
- [ ] 圆点从正常态 → 激活态（平滑过渡）

## 🚀 快速验证流程

### Step 1: 视觉检查（2分钟）
1. 打开 `http://localhost:5173/analysis`
2. 检查垂直时间线是否清晰可见
3. 确认 Speech 大字号、Visual/Audio 小字号
4. 确认所有技术参数默认隐藏

### Step 2: 交互测试（3分钟）
1. 点击播放按钮
2. 观察时间圆点自动激活 + 外发光
3. 观察详情微动画展开
4. 点击不同事件，测试跳转功能

### Step 3: 连续播放测试（2分钟）
1. 从头播放整个视频（50秒）
2. 观察圆点连续激活效果
3. 检查是否有卡顿或闪烁
4. 确认激活态过渡流畅

### Step 4: 筛选器测试（1分钟）
1. 点击"风险优先"筛选器
2. 确认只显示中高风险事件
3. 点击"纯字幕"筛选器
4. 确认只显示 Speech 事件
