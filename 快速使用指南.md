## 🚀 如何使用

### 1. **启动服务器**
```bash
# 前端已启动在：http://localhost:5175/
# 后端需要确保运行
```

### 2. **访问分析结果页面**
1. 打开浏览器访问：`http://localhost:5175/`
2. 登录系统
3. 上传视频或选择已分析的视频
4. 进入"分析结果"页面

### 3. **体验证据系统**

#### **点击卡片查看证据**
- 在8个分析卡片上，你会看到证据数量徽章，例如：`身份判定 (6)`
- 点击任意卡片，右侧会滑出证据抽屉
- 抽屉中展示：
  - 综合置信度
  - 证据时间线
  - 详细证据列表（视频/音频/文本）

#### **点击证据跳转视频**
- 在证据抽屉中，点击任意证据项
- 视频会自动跳转到对应的时间点
- 并开始播放该片段

#### **查看视频时间轴标记**
- 在视频播放器下方，你会看到彩色的标记点
- 🔵 蓝色 = 视频证据
- 🟣 紫色 = 音频证据
- 🔵 天蓝色 = 文本证据
- 鼠标悬停在标记点上，会显示详细信息弹窗
- 点击标记点也可以跳转视频

#### **观察实时高亮**
- 播放视频时，接近证据点的标记会自动高亮闪烁
- 白色进度条显示当前播放位置

- ✨ 自动高亮：播放接近时自动闪烁

## 🐛 常见问题

### **Q: 点击卡片没反应？**
A: 确保你在"交互分析"视图模式下，并且已经选择了一个已分析的视频。

### **Q: 抽屉中没有证据？**
A: 当前使用的是Mock数据，确保 `evidenceMockData.ts` 文件正确导入。

### **Q: 视频跳转不准确？**
A: 这是正常的，Mock数据中的时间戳是模拟的，与实际视频内容可能不完全匹配。接入真实算法后会精确。

### **Q: 时间轴标记看不见？**
A: 标记在视频控制条上方，确保视频已加载。如果视频太短，标记点可能重叠。
